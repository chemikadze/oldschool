Решение

Краткие методические указания к решению задач

Задача 1. Маджонг

Решение задачи заключается в непосредственной формализации процесса, описанного в условии. Перед этим необходимо доказать, что от порядка снятия фишек ответ не меняется.

Задача допускает большое количество частичных решений, разделяемых по эффективности.

Задача 2. Дом оригинальности и гармонии

Решение данной задачи основано на знаниях математики и разработке алгоритмов решения математических задач с использованием информатики. Процесс решения задачи представляет собой творческий поиск с постепенной разработкой новых методов построения решений, которые пригодны для все больших значений n.

Задача 3. Цепочка слов

Эффективное решение данной задачи заключается в построении бора из заданных слов и использовании в дальнейшем обхода в глубину для предподсчета времен входа/выхода за O(1), которые затем используются для ответов на вопрос, является ли одна вершина предком другой. Это позволяет за O(n) осуществить поиск максимальной подпоследовательности индексов, которые соответствуют цепочке слов.

Данное решение работает за время O(l+n), где l — сумма длин всех слов.

Задача 4. Забор

В основе правильного решения данной задачи лежит жадный алгоритм. В процессе его использования, не ограничивая общности, будем считать, что в первом отрезке красится последняя доска, поскольку отрезки работы между подходами к цистерне можно менять местами, Решение будет оптимальным, если красить последние k непокрашенных досок в каждом заходе.

Полное решение рассматривает каждый отрезок непокрашенных досок за время O(1).

Задача 5. Окопы и окопи

При решении данной задачи можно выделить два этапа: геометрический и графовый. Геометрический этап заключается в построении графа окопов, где вес ребра — расстояние между окопами. На этом этапе осуществляется поиск минимального расстояния между отрезками. На графовом этапе необходимо реализовать поиск кратчайшего расстояния между двумя вершинами во взвешенном неориетнированном графе.

Задача 6. Планировка кухни

Решение данной задачи основано на использовании динамического программирования.
