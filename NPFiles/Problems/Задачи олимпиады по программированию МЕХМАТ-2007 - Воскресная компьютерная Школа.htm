Задачи олимпиады по программированию "МЕХМАТ-2007"

Практический тур

Задача 1. Показуха (50 баллов)

Забор, стоящий рядом со школой и огораживающий заброшенную стройку, не красили много лет. К приезду чиновника из ГорОНО директор школы решил покрасить те части забора, которые будут видны от входной двери в школу. Требуется опре-делить, какие части забора надо покрасить. Забор представляет собой границу N-угольника (не обязательно выпуклого), сто-роны которого расположены вертикально или горизонтально. Вершины N-угольника в порядке обхода хранятся в массиве A из N точек (3 N 20). Также дается "точка обзора" P, расположенная вне данного многоугольника. Все точки задаются поло-жительными целочисленными координатами (X, Y) (в пикселах). Необходимо закрасить те стороны многоугольника, которые целиком "видны" из точки P (то есть никакая часть закрашиваемой стороны не должна заслоняться другими сторонами). Если сторона видна частично, то ее закрашивать не следует. Число N и массив A допустимо задавать в самой программе; точку обзора P следует указывать при выполнении программы (вводя ее координаты с помощью клавиатуры или щелкая мышью на соответствующей позиции графического поля).

Задача 2. Звезды из звездочек (50 баллов)

Даны два текстовых файла (произвольного размера), в каждом из которых с помощью символов "*" ("звездочка") изобра-жена восьмиконечная звезда (до и после этого изображения файл может содержать пустые строки - см. рис.). Длина каждого луча может меняться в пределах от 1 до 20 символов "*" (не считая центра звезды). Две звезды считаются равными, если при совмещении их центров совпадают все их лучи (перед совмещением допустимо "вращать" звезду на угол 90, 180, 270 градусов, а также "отражать" ее относительно горизонтальной или вертикальной оси). Напишите программу, которая запрашива-ет имена двух текстовых файлов с изображениями звезд и выводит сообщение о том, равны или нет эти изображения (напри-мер, для файлов a.txt и b.txt программа должна сообщить, что звезды равны, а для файлов a.txt и c.txt - что не равны. Теоретический тур

Задача 3. Строки Фибоначчи (40 баллов)

Последовательность строк Fn строится следующим образом: строка F1 содержит единственный символ "A", строка F2 содержит единственный символ "B", а каждая последующая строка Fn получается путем сцепления строк Fn-2 и Fn-1 (в указанном порядке). Например, F3 = F1 + F2 = "AB", F4 = F2 + F3 = "BAB", F5 = "ABBAB", F6 = "BABABBAB". Даны целые числа N и K (3<=N<=45). Определить K-й символ в строке FN. Предполагается, что нельзя целиком разместить в оперативной памяти строки FN для больших чисел N из указанного диапазона.

Задача 4. Делитель двойного факториала (30 баллов)

Даны два положительных целых числа N и K; значение N лежит в диапазоне 3-200, значение K может быть произвольным. Проверить, делится ли "двойной факториал" числа N на число K. Двойной факториал числа N определяется следующим обра-зом: N!! = N · (N - 2) · (N - 4) · : (последний сомножитель равен 2 для четных N и 1 для нечетных N).

Задача 5. SMS-ки (30 баллов)

Дана строка S длины N (0 < N < 100), содержащая информацию о том, какие буквенные клавиши мобильного телефона бы-ли нажаты. К сожалению, длительность пауз между нажатиями неизвестна, поэтому, например, строка S, равная "АААА", может означать не только четыре буквы "А", но и одну букву "Г", или две буквы "Б", или букву "Б" и две буквы "А", и т. д. На данном телефоне все буквы сгруппированы в алфавитном порядке по четыре (буква "Ё" отсутствует): АБВГ, ДЕЖЗ, ИЙКЛ, МНОП, РСТУ, ФХЦЧ, ШЩЪЫ, ЬЭЮЯ. Строка S содержит только первые буквы указанных четверок (эти буквы вы-делены жирным шрифтом). Буквы набраны в Windows-кодировке, используются только заглавные буквы. При нажатии кла-виш никогда не происходит циклический возврат к первой букве четверки (например, последовательность "ААААА" не может обозначать единственную букву "А"). Напечатать в алфавитном порядке все комбинации букв, которые можно получить из строки S, набирая ее символы с различными паузами на мобильном телефоне. Например, для строки "ААААА" всеми воз-можными вариантами (в алфавитном порядке) являются ААААА, АААБ, ААБА, ААВ, АБАА, АББ, АВА, АГ, БААА, БАБ, ББА, БВ, ВАА, ВБ, ГА, а для строки "ИИММ" - ИИММ, ИИН, ЙММ, ЙН.
