Интернет-олимпиада по информатике

На сайте http://info-online.rusolimp.ru началась регистрация участников интернет-олимпиады заключительного этапа Всероссийской олимпиады школьников по информатике, которая пройдет с 20 по 26 апреля 2007 года.

Методическая комиссия по информатике Центрального оргкомитета Всероссийской олимпиады школьников сообщает, что в период проведения заключительного этапа XIX Всероссийской олимпиады школьников по информатике, с 20 по 26 апреля 2007 года, любой школьник России сможет принять участие в интернет-турах этой олимпиады, что позволит каждому желающему, независимо от места его проживания, оценить свои возможности при решении олимпиадных задач в режиме реального времени. Интернет-туры будут проводиться по московскому времени в полном соответствии с Правилами проведения заключительного этапа в г.Челябинске по следующему расписанию:

22 апреля – пробный тур, начало в 09.30, окончание в 11.30

23 апреля – первый тур, начало в 08.30, окончание в 13.30

25 апреля – второй тур, начало в 08.30, окончание в 13.30

Порядок участия в интернет-турах заключительного этапа Всероссийской олимпиады школьников по информатике следующий:

1. В интернет-турах олимпиады могут принять участие любые школьники, имеющие компьютеры, подключенные к сети Интернет. Для участия в интернет-турах олимпиады необходимо до начала пробного интернет-тура зарегистрироваться на сайте http://info-online.rusolimp.ru/

2. Сразу после начала интернет-туров олимпиады любой зарегистрированный участник может войти в систему под выбранными во время регистрации логином и паролем.

3. По истечении времени, отводимого на решение задач тура, прием решений участников прекращается.

4. Результатом решения олимпиадной задачи может быть либо исходный текст решения на одном из разрешенных языков программирования, либо выходные файлы для заданного набора входных файлов, о чем сообщается в условии задачи. Разные задачи можно решать с использованием разных языков программирования.

5. Участник может направить на проверку до 5-ти решений (т.е. исходных файлов) каждой из задач. Для оценивания решения задачи участника будет использоваться только последнее решение, посланное им на проверку в течение тура и прошедшее предварительное тестирование для теста из условия задачи.

6. Проверка решений участников будет осуществляться в автоматическом режиме с использованием специализированной проверяющей программной среды. Набор тестов и критерии оценки решений для каждой задачи утверждаются жюри заключительного этапа Всероссийской олимпиады школьников по информатике до начала процесса проверки решений участников. Если решением задачи является программа, то оценка правильности решения осуществляется путем исполнения программы с заранее подготовленными и неизвестными участникам входными файлами с последующим анализом получаемых в результате этого выходных файлов. Если решением задачи является набор выходных файлов для заданного в условии задачи набора входных файлов, то оцениваются только представленные на проверку выходные файлы.

7. Баллы за каждую задачу начисляются по сумме баллов за правильные решения, полученные для каждого теста из набора тестовых входных данных, используемых при проверке соответствующей задачи.

8. Результаты решения участников интернет-туров олимпиады размещаются на сайте соревнований через некоторое время после окончания проверки всех решений участников.
